Lucky 7 Game Instructions

This game involves pressing buttons to generate random numbers. Your goal is to complete 3 rounds and win by getting the same number twice through random selection.

Button Details

Below Button This button generates a random number between 1 and 6.

Lucky 7 Button This button always generates the number 7.

Above 7 Button This button generates a random number between 8 and 12.

How to Play

Start the Game:When the game begins, press one of the three buttons (Below, Lucky 7, or Above 7) to generate your first number.

Rounds

The game has 3 rounds. In each round, you need to press a button to generate a random number. You can select any button in each round: Below 7, Lucky 7, or Above 7.

Winning Condition

If you get the same number twice in any of the 3 rounds, you win! If no number repeats after 3 rounds, the game is over, and you lose.
